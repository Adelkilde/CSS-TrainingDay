Om CSS

CSS er en del af den hellige trilogi:

Hvor HTML tager sig af strukturen, og JavaScript tager sig af funktionalitet og interaktivitet, så håndterer CSS udseendet, hvordan siden skal styles.

Selectorer

Nogle af de grundlæggende selectorer i CSS:

tagname
HTML-navnet - for eksempel section for <section>.
#
id - for eksempel #about for id="about".
.
klasse - for eksempel .codebox for class="codebox".

Lær og øv flere selectorer på CSS Diner.

Box-modellen

Alle HTML elementer er "pakket" ind i en boks, med margin, border og padding.

Margin
Border
Padding
Content

Størrelsen på boksen bestemmes af content - altså mere tekst skubber til boksens størrelse.
Bortset fra border er lagene "usynlige" - eller det vil sige, margin er samme farve som parents baggrund, og padding er samme farve som boksens egen baggrund.

Enheder

Når man skal angive afstande i sit design, kan man bruge flere forskellige enheder

De mest populære (som vi skal bruge) er:

px
pixels - den bruger vi kun når vi er nødt til at være meget præcis
%
procent - den er altid god, tilpasser til % af elementets oprindelige størrelse, eller den contentboks det vises i
vw
viewport width - 1vw svarer til 1% af vinduets bredde, 100vw er hele vinduets bredde. Den er effektiv til at tilpasse til skærmstørrelser.
vh
viewport height - 1vh svarer til 1% af vinduets højde, 100vh er hele vinduets højde. Effektiv når vi vil have noget til at fylde skærmen ud.
em
skriftstørrelsen - 1em svarer cirka til bredden af et M i den valgte skrift. Rigtig god når man vil lave afstande rundt om tekst, der passer til det aktuelle zoom-niveau
rem
dokumentets skriftstørrelse - den skriftstørrelse der er sat på <html> elementet. Til forskel fra em så varierer den ikke fra fx h1 til p.

Husk at 0 - altså ingen afstand - ikke behøver nogen enhed. 0kr. er trods alt det samme som 0€.
